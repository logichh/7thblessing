# 7thblessing

`7thblessing` is an all-in-one premium gameplay + monetization plugin for Paper servers.

It combines timed blessings, subscriptions, boosts, cosmetics, rank perks, and admin tooling into one system with menu-driven UX.

## What is it

- Full Heavenly Blessing system (item activation, timed duration, stacking support, restart-safe timers)
- Ascended Subscription system (timed access, daily key claim flow, perk toggles)
- Global Boost system with broadcasted activation/end states
- Extended Ender Chest system with filters and admin viewing (`/endsee`)
- Collectables and cosmetics unlock tracking
- Rank features system (snapshots, waypoints, theme presets, portfolio, rank tags)
- Upkeep system (monthly goals, contributions, leaderboard, finalize + rewards)
- Store and guide menu framework for server monetization flows
- In-game admin GUI for managing major systems

## Core Gameplay Systems

- Blessings: activate blessing items to grant temporary premium benefits.
- Subscription: grant/revoke timed subscriber access with perk control.
- Boosts: run server-wide boosts (XP, loot, sell, haste, harvest, fishing, builder, explorer).
- Explorer boost protection: can be configured to reward only non-player-placed blocks.
- Ender Chest suite: expanded storage support, filter categories, and staff inspection tools.
- Utility perks: nick, fly, speed, craft, hat, anvil, grindstone, smithing, back, feed, timeout.

## Player-Facing Menus

- Blessings hub
- Subscription menu
- Collectables menu
- Store menu
- Guide menu
- Rank perks and rank tags menus
- Personal systems: warps, aliases, teleport history, loadouts, presets, wardrobe, prestige

## Admin and Economy/Store Systems

- Blessing/admin command suite for granting items, boosts, ranks, cosmetics, subscriptions
- Monthly upkeep management with top contributor rewards
- Command-template friendly monetization flow (works well with Tebex actions)
- Permissions-ready structure for staff and donor access tiers
